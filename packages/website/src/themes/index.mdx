## Usage

Installation
```
yarn add @gio-design/themes
```

JavaScript
```
import { indigo } from '@gio-design/themes';

...
<div style={{ color: indigo.colorPrimary }}>
...
```

React Context
```
export const ThemeContext = React.createContext({});
```
```
import { indigo } from '@gio-design/themes';
...
  return (
    <ThemeContext.Provider value={indigo}>
...
```


```
...
  <ThemeContext.Consumer>
    {
      (theme) => (...)
    }
  </ThemeContext.Consumer>
...
```
或使用 useContext
```
...
  const theme = useContext(ThemeContext);
...
```

Webpack
```
var indigo = require('@gio-design/themes/lib/indigo-less');

...
{
  loader: 'less-loader',
  options: {
    'modifyVars': indigo
  }
}
...
```

umiJS

.umirc.ts
```
const indigo = require('@gio-design/themes/lib/indigo-less');
...
theme: indigo
...
```

Pacel
```
echo "{\"modifyVars\": $(cat node_modules/@gio-design/themes/lib/[THEME_NAME]-less.json) }" > .lessrc
```
