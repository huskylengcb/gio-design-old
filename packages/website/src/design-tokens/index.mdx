# Design Tokens

import Tokens from './Tokens';

## Usage

```
yarn add @gio-design/tokens
```

Less
```
@import '~@gio-design/tokens/dist/tokens.less'; // less-loader

h1 {
  background-color: @color-primary;
}
```

JavaScript
```
import tokens from '@gio-design/tokens';

let color = tokens.colorPrimary;
```
