# Modal

## 简介

用于展示数字的 Icon，1 - 8 都有默认的颜色和背景颜色，如果需要修改可以自行提供

## API

参数 说明 类型 默认值 版本

| 参数            |    说明    |  类型  | 默认值 | 版本 |
| --------------- | :--------: | :----: | :----: | ---: |
| color           | 数字的颜色 | string |
| backgroundColor |  背景颜色  | string |
| number          |    数字    | number |
