# Ellipsis

## 简介

自动按照父元素宽度渲染文字。如果文字宽度大于父元素宽度，给ellipsis并添加tooltip。
支持生命周期动态变化ellipsis。

## Examples
<Playground>
  <div style={{width: '20px'}}>
    <Ellipsis> testttttttttttttttt</Ellipsis>
  </div>
</Playground>
