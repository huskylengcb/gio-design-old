# Developers

配置 `.yarnrc`
```
"@gio-design:registry" "[private_registry]"
```

安装对应的包
```
yarn add gio-design/[package_name]
```

更新比较快，请及时升级
可使用 `yarn outdated` 查看版本情况

### Package List
1. [@gio-design/tokens](/design-tokens)
2. [@gio-design/themes](/themes)
3. [@gio-design/icon](/icons)
4. [@gio-design/components](/components)
